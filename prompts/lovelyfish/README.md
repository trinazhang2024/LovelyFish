# LovelyFish AI Development Prompts

This folder contains the prompts used during the responsive UI refactoring of the LovelyFish e-commerce project.

Phase 1
- Navbar
- Carousel
- Footer

Phase 2
- Homepage
- Product Cards
- Product Grid
- New Arrivals
- Clearance

Phase 3
- Product Detail
- Cart
- Checkout

These prompts are designed for OpenAI Codex and focus on:
- Bootstrap 5
- Responsive Design
- CSS-first approach
- Minimal JSX changes
- Preserve existing business logic

General Rules

- Preserve all existing functionality.
- Do not modify any application logic.
- Do not change React state management.
- Do not modify API calls.
- Keep JSX changes as small as possible.
- Prefer CSS changes over JSX changes.
- Keep Bootstrap 5.
- Do not introduce new libraries.
- Run npm run build after implementation.
- Report all modified files before applying changes.


Requirements:

- Keep Bootstrap 5.
- Preserve all existing functionality.
- Do not modify business logic.
- Do not modify backend APIs.
- Keep the existing project structure.
- Keep external CSS files.
- Reuse existing components wherever possible.
- Minimize JSX changes.
- Prefer CSS changes wherever possible.
- Do not refactor existing components unless absolutely necessary.
- If a JSX change is required, keep it minimal and explain why.
- Show the list of files before making any changes.
- Wait for my approval before editing.
